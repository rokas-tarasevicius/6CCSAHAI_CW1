# Coursework 1

A Streamlit application for uploading PDF files.

## Setup

1. Install uv (if not already installed):
   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```

2. Install dependencies:
   ```bash
   uv sync
   ```

3. Run the application:
   ```bash
   uv run streamlit run app.py
   ```

## Usage

Open the application in your browser and use the file uploader to upload PDF files.
